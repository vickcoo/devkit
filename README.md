## ğŸ“– Usage

### Interactive Mode
```bash
npx @devkit/cli init
```

Follow the prompts to:
1. Select a template
2. Enter project name
3. Choose directory
4. Install dependencies (optional)

### Direct Mode
```bash
npx @devkit/cli init --template nextjs-tailwind --name my-app
```

### List Templates
```bash
npx @devkit/cli list
```

### Configuration
```bash
npx @devkit/cli config
```
